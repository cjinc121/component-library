CSS Flexbox for beginners by beginner

26.9.2021-CSS,FLEXBOX

The Flexbox Layout aims at providing a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic .

Basics :

The flexbox layout involves lot of properties some are applied on the parent element or flex-container and others are meant for children element or flex-items. Also layout of flexbox is done in refernce to two axis:

  • main-axis
  • cross-axis

Flex Properties:

Flex properties are of two types :

  • Parent Properties:
    • display:flex

      This defines a flex container.It enables a flex context for all its direct children.

    • flex-direction: row | row-reverse | column | column-reverse

      This establishes the main-axis, thus defining the direction flex items are placed in the flex container.They are laid out in single direction primarily, either in horizontal rows or vertical columns.

    • flex-wrap: nowrap | wrap | wrap-reverse

      By default, flex items will all try to fit onto one line. You can change that and allow the items to wrap as needed with this property.

    • justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly | start | end | left | right ... + safe | unsafe

      This defines the alignment along the main axis. It helps distribute extra free space leftover when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size.

    • align-items: stretch | flex-start | flex-end | center | baseline | first baseline | last baseline | start | end | self-start | self-end + ... safe | unsafe

      This defines the default behavior for how flex items are laid out along the cross axis on the current line. Think of it as the justify-content version for the cross-axis

  • Children Properties:
    • order :

      By default, flex items are laid out in the source order. However, the order property controls the order in which they appear in the flex container.

    • flex-shrink:

      This defines the ability for a flex item to shrink if necessary.

    • flex-grow:

      This defines the ability for a flex item to grow if necessary. It accepts a unitless value that serves as a proportion. It dictates what amount of the available space inside the flex container the item should take up.

    • align-self: auto | flex-start | flex-end | center | baseline | stretch

      This allows the default alignment (or the one specified by align-items) to be overridden for individual flex items.
